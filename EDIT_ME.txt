HOW TO EDIT YOUR SITE
─────────────────────
Everything you need to change is inside  index.html
at the very top of the file (the first ~250 lines).

1. Open  index.html  in TextEdit (or any text editor)
2. The editable section starts right after the line:
       ✏️  LEXIN'S PORTFOLIO — EDIT YOUR CONTENT HERE
3. Edit the text between the " " quote marks
4. Save the file  (Cmd+S)
5. Open or refresh  index.html  in your browser — done!

RULE: Only change text inside " " quotes.
Don't delete commas  ,  colons  :  or brackets  { }
